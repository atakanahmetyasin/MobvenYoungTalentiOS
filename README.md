<h1 align="center">
     Movie App
</h1>

## Technologies and Libraries
- SwiftUI
- MVVM
- Protocol Oriented Programming
- Async Await
- SwiftData
- SwiftLint
- Unit Tests

## App Features
- Search for movies
- View movie details, images, videos, cast and crew.
- Add movies to favorite.
